4-bit:
i4004 - первый процессор Intel.
i4040 - улучшенная версия предшественника.
8-bit:
i8008 - первый восьми битный процессор Intel.
i8080 - усовершенствованная версия i8008.
i8085 - усовершенствованная версия i8080.
16-bit:
i8086 (iAPX86) - начало архитектуры x86.
i8088 - основан на i8086, но имеет шину данных 8-bit.
i80186 - усовершенствованный версия i8086.
i80188 - основан на i80186, но имееат шину адреса 8-bit
i80286 - усовершенствованный версия i8086
32-bit (RISC):
iAPX 432 - первый тридцатидвух битный процессор Intel.
i80960 - RISC-процессор, популярный в 1990-х годах.
i80860 (N10) - попытка компании сделать новую систему команд.
32-bit (i386):
i80386DX - Первый процессор семейства 386.
i80386SX - решение для ПК начального уровня и портативных ПК.
i80376 - процессор для встраиваемых систем.
i80386SL - первый энергоэффективные для портативных ПК.
i80386EX - модификация 386SX для встраиваемых систем.
32-bit (i486):
i80486DX - Первый процессор семейства 486
i80486SX - i486DX без сопроцессора.
i80486GX - низковольтный вариант i486SX.
i80486DX2 - улучшенный вариант i486DX.
i80486SL - энергосберегающий вариант i486DX для мобильных ПК.
i80486DX4 - улучшенная версия 486DX2.
